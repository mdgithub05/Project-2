# Retail Business Performance & Profitability Analysis

## 📌 Overview

This project analyzes retail business data to evaluate overall performance and profitability. It helps identify sales trends, profit drivers, and areas for business improvement.

## 🎯 Objective

* Analyze sales and profit performance
* Identify top-performing products and regions
* Provide data-driven business insights

## 🛠️ Tools & Technologies

* SQL (Retail Queries)
* Excel / CSV
* Tableau (Dashboard)

## 📁 Project Files

* **Retail Queries** → SQL queries used for data extraction and analysis
* **Retail Report** → Detailed report with insights and findings
* **Tableau Dashboard** → Interactive dashboard for visualization

## 📊 Key Analysis

* Sales & profit trends
* Category and sub-category performance
* Region-wise analysis
* Customer behavior insights

## 📈 Outcome

* Identified profitable segments
* Highlighted loss-making areas
* Supported better business decision-making

## 🚀 Conclusion

This project demonstrates how data analysis and visualization can improve retail business performance and profitability.
